# Palindrome Checker

## Created by: Eric Stratton, Joey Guinta vs. 12/08/2020

## Description

Take a word input by the user and check to see if it's a palindrome.

## Specs

- Take word from user input
- Reverse word
- Check if the word reversed is equal to the original word
